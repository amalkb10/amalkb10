# 🚀 GitHub Developer Portfolio Dashboard

A modern, responsive GitHub-inspired developer portfolio dashboard built using HTML, CSS, JavaScript, and Chart.js. This project showcases developer statistics, contribution activity, programming language distribution, skills, pinned repositories, and recent development activity in an elegant dark-themed UI.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/github-portfolio-dashboard.git
```

### Navigate to Project

```bash
cd github-portfolio-dashboard
```

### Run Project

Simply open:

```bash
index.html
```

or use VS Code Live Server:

```bash
Right Click → Open with Live Server
```
